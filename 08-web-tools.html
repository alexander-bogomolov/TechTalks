Grunt - Gruntfile.js


  module.exports = function (grunt) {
    // Project configuration
    grunt.initConfig({
      pkg:     grunt.file.readJSON('package.json'),

      // Compass
      // Define configuration for the compass module: grunt-contrib-compass
      compass: {
        dev:  {
          options: {
          config: 'compass-config-dev.rb'
        }
      },
    });

    // Load modules
    grunt.loadNpmTasks('grunt-contrib-compass');

    // Register Tasks
    grunt.registerTask('css-dev', ['compass:dev']);
  };

  1. Theme-Starter

Commands

  • compass:dev - Kompeliert Compass im "dev"-Modus. (Ausgedehnt und mit Kommentaren.)
  • compass:prod - Kompeliert Compass im "prod"-Modus. (Komprimiert, minifizeirt und ohne Kommentare.)
  • jshint:beforeconcat - Überprüft JavaScrit Dateien auf richtigkeit.

Commands

  • qunit - Lässt Unit Tests laufen. (Wenn welche definiert sind.)
  • concat - Führt alle JavaScript Dateien zusammen.
  • uglify - Minifiziert die zusammengeführte JavaScript Datei.
  • htmlhint:build - Validiert das HTML

Tasks

  • css-dev - ['compass:dev']
  • css-prod - ['compass:prod']
  • js-dev - ['jshint:beforeconcat', 'concat']
  • html - ['htmlhint:build']

Tasks

  • js-prod - ['jshint:beforeconcat', 'concat', 'uglify']
  • default - ['compass:dev', 'jshint:beforeconcat', 'concat']
  • prod - ['compass:prod', 'jshint:beforeconcat', 'concat', 'uglify']
