CSS3 Basics

Einstieg in die moderne Web-Entwicklung

Rahmen

  • border-color - Rahmen als Verlauf
  • border-image - Rahmen aus Bildern
  • border-radius - abgerundeten Ecken
  • box-shadow - Schatten

Hintergründe

  • background-size - die Größe der Hintergrundbilder angeben
  • Multiple Backgrounds - Hintergründe aus mehreren Bildern zusammensetzen
  • Gradients - Verläufe

Farben

  • opacity: 0.6
  • hsl(0, 100%, 50%)
  • hsla(0, 100%, 50%, 0.2)
  • rgba(255, 0, 0, 0.2)

Texte

  • text-shadow - Text mit Schatten
  • text-overflow - Wenn der Text nicht in den Container passt, dann...
  • word-wrap - Umbrechen erzwingen

Selektoren

  • E[foo] - Element E mit Attribut foo
  • E[foo="bar"] - Element E mit Attribut foo und Wert bar
  • E[foo~="bar"] - Element E mit Attribut foo, dessen durch Leerzeichen getrennte Liste von Werten bar enthält (p[class~="bar"] entspricht p.bar)

Selektoren

  • E[foo|="bar"] - Element E mit Attribut foo, dessen durch Trennstriche (-) getrennte Liste von Werten mit bar beginnt
  • E[foo^="bar"] - Element E mit Attribut foo, dessen Wert mit bar beginnt

Selektoren

  • E[foo$="bar"] - Element E mit Attribut foo, dessen Wert mit bar endet
  • E[foo*="bar"] - Element E mit Attribut foo, das die Zeichenkette bar enthält
  • Viele mehr!
  1. http://kimblim.dk/css-tests/selectors/
  2. http://jendryschik.de/wsdev/einfuehrung/css/selektoren

Multi-column

  • column-count - Container in Spalten unterteilen
  • column-gap - Abstand zwischen den Spalten
  • column-rule - Trennlinie zwichen den Spalten

Media Queries

/* smartphones in landscape mode */
@media screen and (min-width: 480px) {}

/* tablets in portrait mode */
@media screen and (min-width: 768px) {}

/* tablets in landscape mode */
@media screen and (min-width: 1024px) {}

/* standard laptop screens */
@media screen and (min-width: 1220px) {}

/* bigger desktop screens */
